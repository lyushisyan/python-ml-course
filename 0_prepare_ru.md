## Вариант 1: Официальный Python + pip

> **Целевая аудитория:** Инженеры-программисты, веб-разработчики.   
> **Особенности:** Самый чистый, полностью настраиваемый, занимает минимум места.

#### 1. Установка
1. Перейдите на страницу загрузки [официального сайта Python](https://www.python.org/downloads/).
2. Скачайте и запустите установщик.
3. ⚠️ **Важный шаг:** Обязательно поставьте галочку **"Add Python to PATH"** внизу окна.
4. Нажмите **Install Now**.

#### 2. Настройка (CMD)
Откройте командную строку (CMD) и выполняйте следующие команды по порядку:

1. Проверка установки
```
python --version
```

2. Создание виртуальной среды (рекомендуется создавать отдельно для каждого проекта)
```
python -m venv my_env
```

3. Активация среды (после успешной активации в начале строки появится (my_env))
```
my_env\Scripts\activate
```

4. Обновление pip
```
python -m pip install --upgrade pip
```

5. Установка основных библиотек
```
pip install numpy scipy matplotlib
```

```
pip install jupyter notebook
```

6. Запуск Jupyter Notebook
```
jupyter notebook
```

## Вариант 2: Miniconda

> **Целевая аудитория:** Аналитики данных, исследователи.   
> **Особенности:** Включает менеджер пакетов Conda, средний размер, мощное управление средами.

### 1. Установка

1. Перейдите на сайт: [https://docs.conda.io/en/latest/miniconda.html](https://docs.conda.io/en/latest/miniconda.html)
    
2. Скачайте установщик Windows 64-bit и установите (можно оставить настройки по умолчанию).

### 2. Настройка (Anaconda Prompt)

В меню «Пуск» найдите и откройте Anaconda Prompt (не используйте CMD), затем выполните следующие команды:

1. Создание среды
```
conda create -n numcalc python=3.12
```

2. Активация среды
```
conda activate numcalc
```

3. Установка библиотек
```
conda install numpy scipy matplotlib -y
```

```
conda install jupyter notebook
```

4. Запуск Jupyter Notebook
```
jupyter notebook
```

## Вариант 3: Anaconda

> **Целевая аудитория:** Начинающие, студенты, пользователи без технического бэкграунда.  
> **Особенности:** «Всё включено», сотни предустановленных библиотек, не требует настройки, огромный размер установочного файла (3 ГБ+).

### 1. Установка

1. Перейдите на сайт: [https://www.anaconda.com/download](https://www.anaconda.com/download)

2. Скачайте и установите (процесс установки долгий, пожалуйста, подождите).

### 2. Запуск (Графический интерфейс)

Ввод кода не требуется, запуск через графический интерфейс:
- Нажмите меню Windows «Пуск».
- Найдите папку Anaconda3.
- Нажмите на иконку Jupyter Notebook.
- Браузер откроется автоматически, можно начинать писать код.
